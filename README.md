# AA Trading Website

React + Vite + Tailwind CSS product website powered by Airtable CMS.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Airtable API

## Setup

```bash
npm install
npm run dev
````
Environment Variables

Create a .env file in the root:
```bash
VITE_AIRTABLE_BASE_ID=xxxx
VITE_AIRTABLE_TABLE_NAME=xxxx
VITE_AIRTABLE_PAT=xxxx
````
